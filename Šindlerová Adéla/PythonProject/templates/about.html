O projektu

Projekt je rozdělený na 3 části: desktop aplikace, web a databázi. Hra Snake je rozšířená o vlastní prvek: power-upy (speed/slow/poison).

Desktop hra (pygame) Web Flask SQLite REST API

📱 Aplikace

Desktop hra v Pythonu (pygame). Po spuštění se uživatel přihlásí. Pokud účet neexistuje, aplikace vyzve k registraci na webu a otevře stránku registrace.

  • Login přímo v aplikaci
  • Po hře uložení skóre do DB přes API
  • Power-upy jako vlastní rozšíření

Spuštění

python app.py (server)
python desktop_game.py (desktop hra)

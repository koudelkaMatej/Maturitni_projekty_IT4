🎮 Aplikace (desktop hra)

Hra běží jako desktop aplikace. Přihlášení je přímo v aplikaci. Pokud účet neexistuje, aplikace nabídne registraci přes web.

Jak spustit aplikaci

  1. Spusť server: python app.py
  2. Spusť hru: python desktop_game.py

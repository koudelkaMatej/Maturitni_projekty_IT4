O hře

Simulátor záchranářských složek napsaný v Pythonu a Pygame

🎯

Cíl hry

Řídíš tři záchranářské jednotky — policii, hasiče a sanitku. Na mapě města se náhodně objevují události, které musíš vyřešit správnou jednotkou dřív, než vyprší čas.

⚡

Herní mechaniky

Každý typ události vyžaduje jiný typ záchranáře. Pošleš-li špatnou jednotku, vrátí se bez výsledku. Vyřešíš-li událost včas, získáš +1 bod. Pokud událost vyprší bez vyřešení, přijdeš o 1 bod.

🏆

Online žebříček

Po každé hře zadáš jméno a skóre se odešle na školní server. Porovnej se s ostatními hráči v živém žebříčku.

⚙️

Technologie

Python 3 + Pygame 2. Backend v PHP + MySQL na školním serveru. Objektové programování, herní smyčka, síťová komunikace.

Jak hrát

1

Vyber jednotku

Klikni na jeden z panelů základny dole — Policie, Hasiči nebo Sanitka. Panel se zvýrazní žlutě.

2

Klikni na událost

Na mapě se objeví pulzující barevné kruhy — události. Klikni na ten, pro který máš správnou jednotku.

3

Čekej na vyřešení

Jednotka odjede na místo. Dorazí-li správná jednotka včas, událost zmizí a získáš +1 bod. Pozor — pokud čas vyprší dřív, přijdeš o 1 bod!

4

Odešli skóre

Po 1 minutě se hra ukončí. Zadej jméno a odešli výsledek na online žebříček.

Struktura databáze pro ukládání skóre

score_tabulka
🔑 id (INT, PK, AUTO_INCREMENT)
👤 hrac (VARCHAR 50, NOT NULL)
🏆 body (INT, NOT NULL, DEFAULT 0)
📅 datum (DATETIME, DEFAULT NOW())

SQL pro vytvoření tabulky:

CREATE TABLE score_tabulka (
  id    INT AUTO_INCREMENT PRIMARY KEY,
  hrac  VARCHAR(50) NOT NULL,
  body  INT NOT NULL DEFAULT 0,
  datum DATETIME DEFAULT CURRENT_TIMESTAMP
);

Příklady dotazů:

-- TOP 10 hráčů
SELECT * FROM score_tabulka
ORDER BY body DESC LIMIT 10;

-- Vložení skóre
INSERT INTO score_tabulka (hrac, body)
VALUES ('Ondrej', 15);

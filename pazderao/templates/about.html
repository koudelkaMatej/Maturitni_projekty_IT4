MATURITNÍ PROJEKT

Hra Breakout

Tento projekt vznikl jako praktická maturitní práce. Jedná se o klasickou arkádovou hru Breakout naprogramovanou v jazyce Python pomocí knihovny Pygame.

Technologie

  • Hra: Python 3, Pygame
  • Web: Flask, HTML, CSS (Jinja2)
  • Ukládání: Textový soubor / MySQL databáze

Zde pak můžeš vložit obrázek svého ER diagramu nebo další popis struktury kódu.

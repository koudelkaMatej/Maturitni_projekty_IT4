O projektu

Popis programu

Tento projekt představuje jednoduchou hru s názvem Flappy Turtle, která byla inspirována známou hrou Flappy Bird. Hráč si může nastavit obtížnost dle sebe a hrát pod svým jménem. Získané skóre si po registraci a přihlášení může zobrazit na webové stránce hry.

Tato hra byla vytvořena v Python využitím Pygame knihovny. Skóre hráče se ukládá do SQLite database a zobrazuje se v leaderboardu.

Webová stránka využívá JavaScript pro interaktivní zoom a posun vývojového diagramu.

Použité knihovny

  • pygame – grafika, zvuk, vstup (externí, pip install pygame)
  • random – náhodné generování (vestavěná)
  • json – ukládání uživatelů (vestavěná)
  • os – kontrola souborů (vestavěná)
  • sqlite3 – databáze skóre (vestavěná)
  • Flask – webové rozhraní (externí, pip install flask)

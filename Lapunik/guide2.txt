Jak aktualizovat (UPDATE):
    cursor.execute("UPDATE users SET highscore = GREATEST(highscore, %s) WHERE id = %s", (score, id))
    connection.commit()

GREATEST() je funkce přímo v SQL – vrátí větší ze dvou čísel.
Takže do DB uložíme vždy jen to VYŠŠÍ skóre, ne vždy to nové.

Bezpečnost hesel – SHA-256 hash:
    import hashlib
    hash = hashlib.sha256("moje_heslo".encode()).hexdigest()
    # výsledek: "5e884898da..."

Jak to funguje při přihlášení:
  1. Uživatel napíše heslo → přepočítáme hash
  2. Porovnáme hash s tím co máme v DB
  3. Pokud sedí → přihlásíme, jinak → chyba

    def test_weapon_level_capped_at_3():

        g.weapon_level = 3
        g.weapon_level = min(3, g.weapon_level + 1)

assert X, "zpráva" = "Ujisti se, že X je pravda. Pokud ne, vyhoď chybu."

Testy jsou rozděleny do 4 sekcí:
  1. Databáze

Barvy v Pygame jsou jako (červená, zelená, modrá) – každá 0–255:
  (255, 0, 0)   = červená
  (0, 255, 0)   = zelená
  (0, 0, 255)   = modrá
  (255, 255, 0) = žlutá
  (0, 255, 255) = cyan
  (255, 255, 255) = bílá
  (0, 0, 0)     = černá

Jak přidat podmínku (IF):
  Příklad: zbraň level 4 = 4 střely
  V game.py, metoda events(), najdi sekci "if k == pygame.K_SPACE":
    elif self.weapon_level >= 4:
        self.bullets += [pygame.Rect(cx-20, top, 4, 12), ...]

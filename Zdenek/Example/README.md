# Tower Defense Lite (Flask + MySQL + Canvas)

## Popis projektu
Jednoduchá tower defense hra v prohlížeči (Canvas) propojená s webovou aplikací (Flask) a databází (MySQL).
Po skončení hry se výsledek odešle na Flask API, uloží do DB a uživateli se přičtou coiny.

## Použité technologie
- Python + Flask
- MySQL (školní server)
- HTML/CSS/JavaScript (Canvas)
